FAQ

Common Sawaal

Jo aksar poochhte hain.

Kya mujhe koi app banana padega?

Bilkul nahi. Aapka shop ek normal website hai jo mobile pe perfectly kaam karta hai. Customers kuch bhi download nahi karte.

JazzCash/Easypaisa kaise connect hoga?

Aap apna JazzCash ya Easypaisa number settings mein add karo. Customers directly aapke number pe payment bhejte hain — hum beech mein nahi aate.

Kya domain name alag se khareedna padega?

Monthly/Quarterly plan mein hum apna domain use karte hain. Yearly plan mein aapke liye custom domain (jaise sweetbakery.pk) setup karte hain.

Agar band karna ho to?

Koi contract nahi, koi lock-in nahi. Jab chahein band karo. Monthly plan mein next month ka sirf payment nahi karna.

Kitne products add kar sakte hain?

Unlimited products. Unlimited categories. Unlimited orders. Koi limit nahi.

Support kaise milega?

Direct WhatsApp pe. Koi ticket system nahi, koi waiting nahi. Seedha contact karo aur help milegi.

Agar internet nahi ho to orders miss hote hain?

Orders database mein save rehte hain. Internet aane par admin panel mein sab dekhega. Koi order miss nahi hota.

Kya multiple branches ke liye alag system chahiye?

Nahi. Ek hi system se multiple branches manage ho sakti hain. Har branch ka apna admin, apne products, apne orders.
